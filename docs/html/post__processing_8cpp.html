double compute_score ( int  N,
const PartitionsMap &  data_dict,
int  makespan 
)

Computes the complexity score based on partitions and timesteps.

This function calculates a score using the time of factorization and applies an exponential weighting factor based on the partition sizes. The result is returned as the logarithm of the computed score.

Parameters
NThe total number of agents.
data_dictA map where each key is a timestep, and the corresponding value is a partition, each partition representing the splitting of the problem.
makespanThe total time duration considered for the computation.
Returns
The logarithm of the computed score.
