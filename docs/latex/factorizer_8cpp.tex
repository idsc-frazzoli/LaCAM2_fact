\doxysubsection{Detailed Description}
Implements various factorization algorithms for multi-\/agent pathfinding. 

This file contains the definitions of the methods declared in the {\ttfamily \mbox{\hyperlink{factorizer_8hpp}{factorizer.\+hpp}}} header file. It includes the logic for different factorization strategies such as distance-\/based, bounding box-\/based, orientation-\/based, A$\ast$-\/based, and definition-\/based factorization algorithms. The implementation details of these strategies are encapsulated in their respective classes, which inherit from the base class {\ttfamily \mbox{\hyperlink{classFactAlgo}{Fact\+Algo}}}. Additionally, utility functions and heuristics for determining agent factorization are provided.
