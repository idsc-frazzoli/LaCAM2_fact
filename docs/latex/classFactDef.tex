Constructor for the {\ttfamily \mbox{\hyperlink{classFactDef}{Fact\+Def}}} class that initializes the object with a given width and handles the initialization of the {\ttfamily partitions\+\_\+map} from the Fact\+Def\+\_\+partitions.\+json file. 

This constructor reads partitioning data from a JSON file located at a predefined path ({\ttfamily assets/temp/\+Fact\+Def\+\_\+partitions.\+json}) and initializes the {\ttfamily partitions\+\_\+map} member variable with it. If the file cannot be opened or a JSON parsing error occurs, it outputs an error message to {\ttfamily std\+::cerr}.
