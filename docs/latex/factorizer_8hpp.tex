\doxysubsection{Detailed Description}
Provides classes and methods for various factorization algorithms. 

This file defines several classes that implement different algorithms for factorizing instances in a multi-\/agent pathfinding problem. Each class inherits from the base class \mbox{\hyperlink{classFactAlgo}{Fact\+Algo}} and provides specific heuristics for determining factorizability.
